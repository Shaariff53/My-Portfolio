# Shaariff Mujtaba - Portfolio

## About

Personal portfolio website for Shaariff Mujtaba - Web Developer, Automation Expert, SEO Specialist & Computer Science Student.

## Technologies Used

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
- Framer Motion

## Getting Started

```sh
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```
